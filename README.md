# 🌿 ZumGarden Frontend Clone (E-commerce Website)

> A pixel-perfect frontend clone of [ZumGarden](https://zumgarden.com/) — a premium wellness and lifestyle e-commerce brand. Built with HTML, CSS, and JavaScript featuring product showcases, video backgrounds, star ratings, promise icons, and a fully structured product page layout.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- jQuery
- WebP / SVG / MP4 Media Assets

---

## 🚀 How to Run

1. Project folder download/clone karo
2. `pro.html` browser mein open karo
3. Done! ✅ No server required — pure frontend hai

## ✨ Features

- 🎬 **Video Hero Section** — Autoplay MP4 background video for immersive experience
- 🛍️ **Product Showcase** — Premium product images with WebP optimization
- ⭐ **Star Rating System** — SVG + PNG based customer rating display
- 🤝 **Brand Promises Section** — Icon-based trust badges (3 promise icons)
- 👒 **Merchandise Gallery** — Hats, apparel aur lifestyle product display
- 🖼️ **Lifestyle Image Grid** — 10+ curated lifestyle photos (pic1–pic10)
- 📱 **Social Media Integration** — Social stack banner for brand presence
- 🌿 **Brand Identity** — ZumGarden yellow & dark color scheme with spore logo
- 🎨 **WebP Optimized** — Fast loading with modern WebP image format
- ⚡ **JavaScript Enhanced** — Interactive UI elements via JS & jQuery
- 📐 **Pixel-Perfect Clone** — Closely matches original ZumGarden.com layout
